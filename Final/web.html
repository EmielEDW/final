Bevindingen / Evaluatie

Tijdens het maken van deze website heb ik terug veel bijgeleerd over webtechnieken. Door deze site te maken weet ik weer bijna alles van html/css want ik was dit toen ik begon een stuk vergeten omdat ik al zo lang geen site's had gemaakt. Ik heb geleerd hoe ik met javascript een slideshow in mijn site zet die images kan cyclen. Ook heb ik geleerd hoe ik een simpel menu maak voor als je de website bekijkt op gsm. Mijn videos heb ik voor het eerst gecompressed zodat ze in mijn site gingen. Ook heb ik mijn site zo goed als responsive gemaakt door display: flex; te gebruiken en row/col te gebruiken. Met behulp van @media heb ik dan de columns onder elkaar gezet als het te klein werd. Wat ik anders zou doen de volgende keer dat ik een site maak is kijken hoe ik het meest overzichtelijk mijn css files opmaak. Want nu heb ik voor iedere pagina een file gemaakt en sommige dingen gedupliceerd wat het vaak onoverzichtelijk maakte.
