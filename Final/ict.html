The Buzz Wire Game

Heb je vroeger ooit dat spel gekregen waar je jouw hand heel stil moest houden en een parcour moest aflopen en als je de draad aanraakte je huis ontploft? Nee, ik ook niet. Maar wie wilt nu dat zijn huis ontploft? Ik heb toch die game gemaakt maar ik heb er speciaal voor gezorgd dat als jij de draad aanraakt jouw huis niet ontploft maar er gewoon een LED gaat branden om jou te laten weten dat je het parcour niet succesvol voltooit hebt.

Maken Van Project

Dit is de uitleg voor een buzz wire game project. Als je dit stappenplan zorgvuldig volgt zal je deze game perfect kunnen namaken bij je thuis en kan je misschien wel iets bijleren. Hieronder staat alles stap voor stap uitgelegd veel succes!

Benodigdheden
1x Arduino
1x Metalen draad
3x Houten plankjes
ca. 8x Male to male connectors
1x Buzzer (optioneel)
1x LED lampje
ca. 2x Resitors 220 ohm

The Source Code

Voor je begint neem je arduino en upload deze code erop.
Deze code zorgt ervoor dat wanneer de handel het parcour aanraakt. Er stroom op komt en het ledje gaat branden.

Code


int handle = 11;

void setup() {
pinMode(handle, OUTPUT);
}

void loop() {
digitalWrite(handle, HIGH);
}

Stap 1

Plooi de draadvorm door een hard metaal in een vorm te buigen die je zelf kan kiezen.

Stap 2

Plak eerst de 2 uiteinden af met plakband en klop ze dan in een plankje hout. Zorg ervoor dat de uiteinden genoeg uitsteken zodat je deze later nog kan verbinden.

Stap 3

Verbind nu een kabel die je verbindt tussen het breadboard en de draad (parcour). Zet de kabel op dezelfde lijn met de anode van de led. Zodat dat als er stroom op het parcour

Stap 4

Maak nu ook de poten van de stand .Dit doe je door 2 extra plankjes toe te voegen aan de base zoals dit.

Stap 5

Nu maak je de handle. Hiervoor maak je eerst een metalen loop met hetzelfde materiaal waarmee je het parcour maakte. Steek deze dan in een stukje (hout, potlood, bamboe). Verbind dan het metaal met een draad naar je arduino op pin 11.

Stap 6

Finish door alle gaten te dichten en alles mooi te maken.

Stap 7

Verbind als laatste de arduino + breadboard met de onderkant van het bord. Zorg dat de 5v en gnd verbonden is met het breadboard. Zorg ook dat de 11de poort . verbonden is met de handle.

Einde

Dit is hoe alle bedrading moet verbonden zijn als je eens een prototype wilt maken: Dit is een vervanging voor het Tinkercad schema omdat dit niet moglijk is om te maken via tinkercad.

Logboek/Planning

Hieronder staat mijn planning voor het maken van het project. De tijdlijn is fictief maar de verschillende activiteiten die in de planning staan heb ik wel allemaal moeten doen.






Slot

Als je het stappenplan zorgvuldig hebt opgevolgd heb je nu normaal een werkende buzzwire game. Ik heb zelf het uitgewerkte project niet kunnen afmaken omdat we andere dingen moesten leren en dit hierdoor niet kon.

Mijn bevindingen

Tijdens het maken van het prototype voor dit project heb ik ondervonden dat ik niet direct mijn handel kon verbinden met 5v om het parcour onder stroom te zetten. Ik moest met behulp van een simpele code de pin waarmee de handel verbonden is onder stroom zetten. Verder heb ik ook geleerd hoe ik codeer in Arduino.

Zelf evaluatie

Ik vindt dat ik veel heb bijgeleerd tijdens het maken van dit project. Maar ik heb vooral veel bijgeleerd van de research die ik ervoor moest doen. Ik vindt dat ik voldoende moeite heb gedaan om mijn project uit te werken en mijn documenten in orde te brengen. Ik denk ook dat ik actief werkte tijdens de lessen en enthousiast was over dit project. Ik vond de opdracht die ik heb uitgewerkt niet heel moeilijk met als reden dat ik het niet volledig kon uitwerken hiermee bedoel ik een definitieve en volledige fysieke versie van mijn project maken.
